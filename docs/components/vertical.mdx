#### 代码示例

此示例是一个三行垂直的 PanelGroup。点击/触摸面板之间的拖拽条并拖动即可调整大小。PanelGroup使用direction属性，值为vertical可设置为垂直面板，默认值是horizontal，即水平面板。

<VerticalDemo />

```jsx
  <PanelGroup direction="vertical">
    <Panel minSize={10} defaultSize={30}>
      left
    </Panel>
    <PanelResizeHandle />
    <Panel minSize={20} defaultSize={30}>
      middle
    </Panel>
    <PanelResizeHandle />
    <Panel minSize={20} defaultSize={40}>
      right
    </Panel>
  </PanelGroup>
```
