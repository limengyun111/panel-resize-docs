#### 代码示例

面板默认会裁剪内容，以避免在调整大小时显示滚动条。不过，可通过修改内容区的overflow属性值，改变内容在面板内溢出的表现。

<OverflowDemo />

```jsx
 <PanelGroup direction="horizontal">
   <Panel>
     <div style={{ overflow: 'auto' }}>
       {/* Content */}
     </div>
   </Panel>
   <PanelResizeHandle />
   <Panel>
     <div style={{ overflow: 'auto' }}>
       {/* Content */}
     </div>
   </Panel>
 </PanelGroup>
```
