#### 代码示例

此示例是一个 3 列水平 PanelGroup。点击/触摸面板之间的空白处并拖动即可调整大小。箭头键也可用于调整面板大小。这些面板使用 minSize 和 defaultSize 属性，defaultSize为整个面板组的默认百分比，minSize为整个面板组的最小百分比。


<HorizontalDemo />


```jsx
  <PanelGroup>
    <Panel minSize={10} defaultSize={30}>
      left
    </Panel>
    <PanelResizeHandle />
    <Panel minSize={20} defaultSize={30}>
      middle
    </Panel>
    <PanelResizeHandle />
    <Panel minSize={20} defaultSize={40}>
      right
    </Panel>
  </PanelGroup>
```
