#### 代码示例

默认情况下，此库会根据用户拖拽过程管理光标。您可以通过PanelGroup提供的customCursor方法覆盖默认光标，customCursor方法接收鼠标是否按下的状态，您可以根据这个状态改变光标样式。但通常不建议自定义光标。

<CustomCursorDemo />

```jsx
  const customCursor = ({ isPointerDown }) => {
    return isPointerDown ? 'grabbing': 'grab'
  }
  return (
    <PanelGroup customCursor={customCursor}>
      <Panel minSize={10} defaultSize={30}>
        left
      </Panel>
      <PanelResizeHandle />
      <Panel minSize={20} defaultSize={30}>
        right
      </Panel>
    </PanelGroup>
  )
```
